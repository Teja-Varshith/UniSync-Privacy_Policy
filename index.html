This Privacy Policy explains how UniSync ("we", "our", or "us") collects, uses, and protects information when you use the UniSync mobile application. UniSync is a student-focused platform that provides tools such as project sharing, portfolio building, interview preparation, and collaboration features.

1. Information We Collect

Account Information

  • Email address (via Google Sign-In)
  • Basic profile information such as name or profile photo

User Content

  • Resumes uploaded by the user
  • Portfolio and project information
  • Interview responses or practice session data
  • Profile information voluntarily provided by the user

Technical and Usage Information

  • Device type and operating system version
  • Application performance data
  • Crash reports used for improving the app

2. How We Use Your Information

  • Authenticate users through Google Sign-In
  • Provide core UniSync features such as portfolios and interviews
  • Display personalized content and recommendations
  • Improve app performance and reliability
  • Maintain platform security and prevent misuse

3. Data Storage and Security

User data may be stored securely using cloud infrastructure and databases. All communication between the UniSync app and servers is encrypted using HTTPS. We take reasonable measures to protect information from unauthorized access.

4. Third-Party Services

UniSync may use trusted third-party services including:

  • Firebase – authentication and notifications
  • Cloud hosting providers for backend services
  • Database services for storing application data

These providers may process limited information according to their own privacy policies.

5. Data Sharing

We do not sell personal information. Data may only be shared:

  • When required by law or legal process
  • To protect the security of the platform
  • With infrastructure providers necessary to operate the application

6. User Data Control

Users can:

  • Update their profile information
  • Delete uploaded content such as resumes or portfolio data
  • Stop all data collection by uninstalling the application

7. Children's Privacy

UniSync is intended for users aged 13 and above. We do not knowingly collect personal information from children under 13.

8. Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Updates will be reflected on this page.
